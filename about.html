About

Static, maintainable LS-DYNA material library + keyword generator. Each model has its own generator module to avoid breaking the entire system when adding new materials.

How to add a new model

  1. Create generator file in /assets/js/generators/ (prefix by MAT for sorting)
  2. Register it in registry.js (import + map)
  3. Add metadata + inputs to /assets/data/materials.json

Design goals

  • Easy maintenance (small modules)
  • Isolation (generator errors won’t break other models)
  • Friendly DYNA workflow (inputs → preview → download)
