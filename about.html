About

Modular LS-DYNA material hub. The core UI never imports generators directly. Each model points to its own generator module via JSON.

Add new model (2 steps)

  1. Add entry in assets/data/materials.json with generatorModule
  2. Create generator file exporting default function generate(params, meta)

Why this won’t break the whole site

  • Dynamic import per model + try/catch
  • Even if generator fails, UI still works
  • No global registry to forget updating
