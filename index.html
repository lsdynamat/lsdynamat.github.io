How to add a new model

  1. Create a generator file in assets/js/generators/ exporting default function generate(params, meta)
  2. Add model metadata + inputs in assets/data/materials.json and point to that module path
  3. Done. No registry edits.

Safety

  • Broken generator only affects itself (try/catch + sandboxed dynamic import)
  • UI never depends on a single JS file registry
  • Clear error messages for debugging
